主线 vs 支线

主线对话（中间栏）

  • 项目全局视角 — 需求讨论、任务拆解、整体调度
  • 用户主入口 — 大部分交互发生在这里
  • 汇总信息 — Agent 状态摘要、进度概览
  • 进入支线的入口 — 点击任务卡片或「对话↗」打开右侧支线

支线对话（右侧栏）

  • 单任务/Agent 视角 — 深入某个具体任务的执行细节
  • 可对话 — 用户可以对 Agent 说"回调记得加幂等校验"
  • 三个 Tab — 对话 / 执行日志（终端流） / 代码变更（diff）
  • 按需打开 — 默认关闭，点击任务卡片或主线中的链接打开

设计要点

三栏布局

  • 左栏 · 任务卡片 — 项目所有任务，带状态和进度，点击切换右侧对话上下文
  • 中栏 · 活动流 — 精简的时间线，快速浏览所有事件（用户指令、AI 规划、Agent 状态变更）
  • 右栏 · 对话面板 — 完整的 AI 对话界面，日志/变更嵌入对话气泡中，底部有输入框

对话中的日志展示

  • Agent 执行日志作为折叠卡片嵌在 AI 回复的气泡中
  • 执行中的 Agent 显示实时终端输出 + 动画指示
  • 完成的 Agent 显示文件变更列表 + 查看代码/日志按钮
  • 用户可以在对话中直接追问：进度、问题、调整

布局特点

  • 左栏任务卡片 — 带状态标记（执行中/待执行/已完成），可筛选，Agent 执行时卡片实时更新进度
  • 右栏对话活动流 — 用户消息、AI 规划建议、Agent 执行日志、完成通知按时间排列
  • 双向联动 — 点击左侧任务卡片，右侧自动定位到相关活动；右侧提到任务时，左侧高亮对应卡片
  • 底部对话框 — 统一入口，输入指令后 AI 的回复和操作都体现在活动流中

AI 时代交互感来源

  • 不再是"点按钮 → 填表单"，而是"对话 → AI 规划 → 确认 → 自动执行"
  • 用户看到 Agent 的实时执行过程（终端输出流式展示）
  • 任务卡片随 AI 执行实时变化状态，不是手动拖拽
  • 与 Cursor 体验一致：可视化 + 对话驱动的混合模式
